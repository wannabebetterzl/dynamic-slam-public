# 20260504 轻度外扩与部位补刀对比

## 对比对象

| 配置 | 说明 | ATE RMSE (m) | RPE RMSE (m) |
| --- | --- | ---: | ---: |
| `SAM3 + fallback` | 当前主线 | `0.016951` | `0.011670` |
| `bodyparts` | 仅加头部/四肢补刀 | `0.017388` | `0.012772` |
| `mild dilate` | `dilate_pixels: 5 -> 7` | `0.016268` | `0.012387` |
| `bodyparts + mild dilate` | 部位补刀叠加轻度外扩 | `0.016835` | `0.012014` |

## 前端统计

| 配置 | mean mask ratio | mean filtered detections | mean foundation |
| --- | ---: | ---: | ---: |
| `SAM3 + fallback` | `0.136234` | `1.110594` | `0.493020` |
| `bodyparts` | `0.136269` | `1.137369` | `0.494236` |
| `mild dilate` | `0.146821` | `1.119907` | `0.497180` |
| `bodyparts + mild dilate` | `0.146860` | `1.147846` | `0.498328` |

## 结论

1. 轻度外扩是有效的。  
   在当前 `walking_xyz` 序列上，`dilate_pixels` 从 `5` 提到 `7` 后，`ATE RMSE` 从 `0.016951` 改善到 `0.016268`。

2. 单独加头部/四肢补刀并没有带来更好的最终精度。  
   `bodyparts` 版本的 `ATE RMSE = 0.017388`，略差于当前主线。

3. 当“部位补刀”叠加“轻度外扩”时，结果优于原始主线，但仍不如“仅轻度外扩”。  
   - `bodyparts + mild dilate`: `0.016835`
   - `mild dilate`: `0.016268`

4. 当前最优选择是：  
   **保留 `SAM3 + fallback` 主结构，并采用保守的轻度外扩，不再额外加部位补刀。**
